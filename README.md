# MoodMix

AI-powered cocktail and beverage recommendation application.

## Features

- AI-based mood-based drink recommendations
- Interactive recipe cards with step-by-step instructions
- Personalized favorites and inventory management
- Beautiful, responsive UI with smooth animations

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server (runs on port 3000)
npm start
```

### Build for Production

```bash
npm run build
```
